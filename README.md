# PUMP Paper Bot Android

Отдельное Android-приложение для paper trading симуляции PUMP/USDT.

## Что делает

- Показывает два графика со свечами, WMA25 и WMA99.
- Основная стратегия: 1000 USDT, покупка/продажа по 4H WMA25/WMA99.
- Экспериментальная стратегия: 1000 USDT, вход/выход по 2H с подтверждением тренда на 4H.
- Фильтрует боковик через ADX, ATR, объём и наклон медленной WMA.
- Сохраняет виртуальный баланс, позицию, последнюю обработанную свечу и журнал сделок на телефоне.
- При запуске или фоновой проверке догоняет пропущенные закрытые свечи и последовательно применяет BUY/SELL.
- Работает без API-ключей и без сервера: используются публичные свечи Binance.

## Ограничение Android

Фоновая проверка через WorkManager запускается примерно каждые 15 минут, но Android не гарантирует точное время запуска. Поэтому приложение хранит последнюю обработанную свечу и при следующем запуске догоняет пропущенный период.

Для более жёсткого постоянного мониторинга нужен foreground service с постоянным уведомлением.
