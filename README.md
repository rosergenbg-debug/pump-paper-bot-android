# PUMP Сигнал V2 для Android

Android-приложение для экспериментальных сигналов и проверки стратегии PUMP/EUR без автоматической торговли.

## Версия 1.4 — дружелюбный интерфейс и готовность сигнала

Приложение использует закрытые 30-минутные свечи PUMP/USDT, EUR/USDT и BTC/USDT, а также публичный funding PUMPUSDT. Цена PUMP/EUR рассчитывается как PUMP/USDT ÷ EUR/USDT.

Два выбираемых профиля:

- **Осторожный**: ищет один трендовый вход. RSI14 был не выше 40 за последние 12 часов и пересёк 45 вверх; PUMP/EUR выше EMA200; BTC выше EMA200; EMA50 BTC растёт; funding PUMP не положительный. Полный выход при +8%.
- **Агрессивный**: ищет осторожный вход и второй вход после шока. Шок — падение свечи не меньше 3% при объёме не меньше 3x и RSI14 не выше 40; затем в течение 18 часов цена возвращается выше EMA200 при RSI не ниже 45 и BTC выше EMA200. Для шок-входа 50% фиксируется при +6%.

Интерфейс показывает готовность условий от **+100** для покупки до **−100** для продажи. На 95 приложение предупреждает звуком и крупной плашкой, на 100 показывает действие. Это степень выполнения правил стратегии, а не вероятность прибыли.

График загружает больше истории, прокручивается пальцем, показывает отдельные колонки осторожного и агрессивного входа, а пунктирная линия отображает продолжение текущего наклона EMA50 как сценарий — не прогноз.

Выходы:

- ценовой стоп 4,4%;
- Осторожный вход: полная фиксация при +8% или через 24 часа;
- вход после шока: 50% при +6%, остаток с безубытком и trailing-stop 4%, либо закрытие через 24 часа.

Backtest оставлен только для PUMP/EUR, позволяет выбрать дату и один из двух профилей, использует ту же логику, вход на открытии следующей свечи, комиссию 0,15% на каждую операцию и проскальзывание 0,05%.

## Важно

- Исторический результат не гарантирует будущую прибыль.
- V2 оставлен только для PUMP: перенос без изменения параметров на BTC, ETH, SOL, BNB, XRP, DOGE, ADA, TRX и AVAX не подтвердил устойчивую прибыль.
- Приложение показывает сигналы. Оно не отправляет реальные ордера на Bitpanda Fusion.
- Сигнал рассчитывается только после закрытия 30-минутной свечи.
