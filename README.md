# PUMP Paper Bot Android

Отдельное Android-приложение для paper trading симуляции PUMP/USDT.

Что делает:

- стартует с виртуального баланса 1000 USDT;
- получает публичные свечи Binance без API-ключей;
- проверяет стратегию WMA25/WMA99 с фильтрами ADX, объёма и ATR;
- сохраняет виртуальный баланс, позицию и журнал последних сделок на телефоне;
- проверяет рынок в фоне через Android WorkManager каждые 15 минут;
- не делает реальные сделки и не подключается к аккаунту биржи.

Ограничение Android: периодическая фоновая проверка не запускается строго секунда-в-секунду. Для более жёсткого постоянного мониторинга нужен foreground service с постоянным уведомлением.
